# 🏃 Endless Runner Game

A fun and simple endless runner game built using React.js.

## 🎮 Features

* Jump mechanics
* Moving obstacles
* Score system
* Game Over + Restart

## 🚀 Live Demo

https://playful-rabanadas-119245.netlify.app/

## 🛠️ Tech Stack

* React.js
* CSS

## 📦 Installation

```bash
npm install
npm start
```
